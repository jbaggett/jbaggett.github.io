This simulates a known-σ one-sample z-test (H₀: μ = 0 vs H₁: μ = δ), the same model as the Power & Error Visualizer. Each study draws n observations, runs the test, and is tallied. The empirical reject-rate converges to the analytic power shown beside it. Set δ = 0 to see the Type I error rate (≈ α).
