Dotplot Editor

Build a dataset by clicking to add and remove points on a number line. Observe how summary statistics respond to changes in the data.

How to use

Add a point
Click anywhere on the number line. The value is rounded to the nearest integer.
Remove a point
Click an existing dot to remove it.
Undo
Click Undo or press Ctrl+Z to reverse the last action.
Presets
Use the Dataset dropdown to load a preset distribution.
Show Boxplot
Toggle on to see a boxplot aligned above the dots on the same axis.
Challenge
Generates a random target boxplot (some with outliers). Build a dataset from scratch that produces the same boxplot. Checkmarks show which parts match.

Things to try

  • Start with the symmetric dataset. Add a single point far from the others (an outlier). Which changed more — the mean or the median? Why?
  • Remove points one at a time from a skewed distribution. When does the mean get close to the median?
  • Build a bimodal distribution from scratch. Where do the mean and median fall?
  • Click Challenge. Can you build a dataset that matches the target boxplot? After you match it, can you build a different dataset with the same boxplot?
