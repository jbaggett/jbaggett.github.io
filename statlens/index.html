How to install as an app

iPhone / iPad (Safari):

  1. Tap the Share button ﻿⬆︎
  2. Scroll down, tap Add to Home Screen
  3. Tap Add

Android (Chrome):

  1. Tap the three-dot menu ⋮ (top right)
  2. Tap Install app or Add to Home Screen
  3. Tap Install

Once installed, StatLens opens full-screen like a native app and works offline.
