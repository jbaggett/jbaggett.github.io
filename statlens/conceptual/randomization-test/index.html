Help

Walk through the logic of a randomization test step by step.

How it works

  1. See the original data as colored cards in two groups
  2. Understand the null and alternative hypotheses
  3. Predict what happens when we shuffle randomly
  4. Shuffle once to see one simulated difference
  5. Repeat many times to build a null distribution
  6. Compare the observed difference to the null distribution

Why cards?

Each card represents one observation. The color is the outcome. Shuffling simulates what would happen if the group assignment didn't matter — this is the core logic behind every randomization test.
